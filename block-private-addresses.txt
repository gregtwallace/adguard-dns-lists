## Block all private addresses
# see: https://en.wikipedia.org/wiki/Reserved_IP_addresses
#
# Provides DNS Rebinding Protection
#
# TODO: ipv6
#
# 192.168.0.0/16 - Class C
/^192.168(.[0-9]{1,3}){2}$/
# 172.16.0.0/12 - Class B
/^172.(1[6-9]|2[0-9]|3[0-1])(.[0-9]{1,3}){2}$/
# 10.0.0.0/8 - Class A
/^10(.[0-9]{1,3}){3}$/
